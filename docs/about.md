## About This Website

This website shares imaginative stories written by **Albert Feijao**.

Many of the ideas come from everyday imagination: dragons, dinosaurs, space adventures, mysterious castles, and curious creatures.

The goal of this small project is simple:

- share creativity
- encourage storytelling
- show how imagination can become written stories

All stories are written in **Markdown** and published using **GitHub Pages**.

---

## Why Publish These Stories?

Stories help develop:

- imagination
- language
- curiosity about the world

By publishing them online, other children, families, and teachers can also read and enjoy them.

---

## Technology Behind the Website

This website is intentionally simple and transparent.

It uses:

- **GitHub Pages**
- **Jekyll**
- **Markdown**
- **jekyll-theme-midnight**

This means the stories are stored as plain text files and the entire website is automatically generated.
